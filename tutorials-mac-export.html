Tutorials - Mac Exporting

This comes up sometimes so I figured I would cover it. When exporting for the Mac, there are a few modifications you must make to the ZIP file to make it work. Sadly, as far as I know, there is no way to automate this during the export process.

First, we need to add some files to the exported game's zip. Inside the .zip/.app/Contents/Resources folder, place the following file:

  • icon.icns
This icon.icns is, obviously, just the icon file for the game using Apple's weird icon extension. You should be able to create it online or with a Photoshop plug-in.

Second, you need to add the following to the .zip/.app/Contents/MacOS folder:

  • libsteam_api.dylib
  • steam_appid.txt
Make sure the libsteam_api.dylib matches the of the Steamworks API version you are using to compile the editor/template or it will crash. The steam_appid.txt file should contain your game's Steam App ID but I think this can be omitted if the game is run directly through Steam itself.

And that's about that for Mac exporting.
