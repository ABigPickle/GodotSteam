GodotSteam A Steamworks module for Godot Engine

Signals For Server

Here is a list of all the currently available signals in the GodotSteam Server module.

_associate_Clan

  • Sent when the game server attempted to be associated with a Steam Group.
  • Emits signal clan_associate in response to function associateWithClan.
  • Sends along the result as an INT; 1 for success and 2 for failure.

_client_Approved

  • Client has been approved to connect to this game server.
  • Emits signal client_approved in response to function sendUserConnectAndAuthenticate, if successful.
  • Sends along the Steam ID of the user wanting to connect and the Steam ID that owns the game. The second will be different if it is borrowed from Family Sharing.

_client_Denied

  • Client has been denied to connection to this game server.
  • Emits signal client_denied in response to function sendUserConnectAndAuthenticate, if failed.
  • Sends along the Steam ID of the user wanting to connect, the reason they were denied as an INT, and any additional text message about the denial.

_client_Group_Status

  • Called when we have recieved the group status of a user.
  • Emits signal group_status in response to function RequestUserGroupStatus.
  • Sends along the Steam ID whose group status is queried, the Steam ID of the group queried, is the user a member as a BOOL, and is the user an officer of the group as a BOOL.

_client_Kick

  • Called when the game server should kick the user.
  • Emits signal client_kick.
  • Sends along the Steam ID of the user being kicked and the reason they were kicked as an INT.

_player_Compat

  • Sent when the game server is checking if the new player is a good fit for the server based on the frenemy system.
  • Emits signal player_compatibility in response to function computeNewPlayerCompatibility.
  • Sends along the result as an INT, number of players that don't like playing with the given player, number of players on the server that the given player doesn't like playing with, number of players in the associated Steam group that don't like playing with the player, and the Steam ID of the given player.

_policy_Response

  • Received when the game server requests to be displayed as secure (VAC protected).
  • Emits signal server_secure.
  • Sends along security response as a BOOL. If true, the game server should display itself as secure to users, false otherwise.

_server_Connect_Failure

  • Server has failed connecting to the Steam back-end. Provides a result reason and possibly retrying.
  • Emits signal server_failure.
  • Sends along the result as an INT and if it is retrying to connect as a BOOL.

_server_Connected

  • Server has connected to the Steam back-end; serverData has no fields.
  • Emits signal server_connected.
  • Sends along no additional data.

_server_Disconnected

  • Called if the client has lost connection to the Steam servers. Real-time services will be disabled until a matching SteamServersConnected_t has been posted.
  • Emits signal server_disconnected.
  • Sends along the result as an INT.
